# 📝 Console To-Do List App

A simple, efficient, and user-friendly Command-Line Interface (CLI) application built with **C#** and **.NET**. This tool helps users manage their daily tasks directly from the terminal with automatic data persistence.

---

## 🚀 Features

- **Add Tasks:** Quickly add new entries to your to-do list.
- **View Tasks:** Display your current list with clear formatting.
- **Remove Tasks:** Delete tasks by their index once completed.
- **Data Persistence:** Tasks are automatically saved to `tasks_data.txt`, so you never lose your progress.
- **Clean Architecture:** Built using clean C# code and organized project structure.

---

- **Framework:** .NET 10.0
